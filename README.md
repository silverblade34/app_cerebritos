# Academia Cerebritos - Aplicativo Móvil para Preparación de Exámenes de Admisión

## Descripción

AcademiaX es una aplicación móvil desarrollada en Flutter, enfocada en brindar a los estudiantes una herramienta integral para prepararse para exámenes de admisión universitaria. Esta aplicación proporciona acceso a exámenes por cursos, simulacros de exámenes de admisión, y análisis detallado del desempeño del estudiante. Además, los usuarios pueden personalizar sus exámenes, acceder a temarios específicos, y recibir recomendaciones de estudio basadas en sus áreas de mejora.

# Instalación y Configuración

## Requisitos Previos
- Asegúrate de tener Flutter instalado. Puedes encontrar instrucciones de instalación en [Flutter Installation Guide](https://flutter.dev/docs/get-started/install).

## Cómo Levantar el Proyecto

1. **Clona el Repositorio:**
    ```bash
    git clone https://github.com/silverblade34/app_cerebritos.git
    ```

2. **Accede al Directorio del Proyecto:**
    ```bash
    cd app_mobile_citvs
    ```

3. **Instala las Dependencias:**
    ```bash
    flutter pub get
    ```

4. **Configura un Emulador con Android Studio:**
   - Abre Android Studio.
   - Crea un emulador desde el AVD Manager (Android Virtual Device).
   - Inicia el emulador.

5. **Ejecuta la Aplicación:**
    ```bash
    flutter run
    ```

La aplicación se abrirá en el emulador que hayas creado.
